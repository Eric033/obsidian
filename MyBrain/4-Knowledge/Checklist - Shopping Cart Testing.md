# Checklist - 购物车通用测试点

## 功能测试
- [ ] 添加商品（正常、库存不足、下架商品）
- [ ] 修改数量（增加、减少、输入非法值）
- [ ] 删除商品（单个、批量、清空）
- [ ] 商品选中/取消（单选、全选）
- [ ] 价格计算（单品、总额、优惠叠加）

## 异常场景
- [ ] 并发操作（多人同时抢购）
- [ ] 登录前后购物车数据合并
- [ ] 价格实时更新（前端展示 vs 后台价格）
- [ ] 库存变化提醒（刷新页面、定时轮询）

## 性能测试
- [ ] 大量商品加载速度
- [ ] 价格计算响应时间
- [ ] 并发提交订单压力测试

## 兼容性测试
- [ ] 多端同步（Web、App、小程序）
- [ ] 浏览器兼容（Chrome、Safari、Firefox）
- [ ] 网络异常（弱网、断网重连）
